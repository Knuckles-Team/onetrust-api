# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-15

### Added
- Initial release: OneTrust API client + MCP server + A2A agent.
- 100% coverage of the OneTrust public API — 35 vendored OpenAPI specs,
  ~600 operations across 7 product areas — generated by
  `scripts/generate_from_openapi.py`.
- 36 action-routed MCP tools (one per domain) plus a `custom_api` escape hatch.
- OAuth2 bearer-token and client-credentials authentication, OIDC delegation,
  multi-region/multi-service host resolution, dual pagination (offset + cursor),
  and `429`/transient-error retry handling.
- Coverage keystone test asserting spec operations == client methods == MCP actions.
